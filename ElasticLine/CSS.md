# CSS Of ElasticLine

- stroke: svg 태그 전용, 텍스트에 색상 효과를 주는 속성

  참고: [w3school](https://www.w3schools.com/graphics/svg_stroking.asp)

- stroke-width: svg 태그 전용, 요소의 선, 텍스트 또는 윤곽선의 두께를 지정

  참고: [w3school](https://www.w3schools.com/graphics/svg_stroking.asp)

- stroke-linecap: svg 태그 전용, 요소의 선의 끝부분 형태를 지정

  참고: [w3school](https://www.w3schools.com/graphics/svg_stroking.asp)
